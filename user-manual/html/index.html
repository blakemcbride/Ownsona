OwnSona ¶

This manual is for OwnSona, a single-user MCP (Model Context Protocol) memory server. OwnSona is built on the Kiss web development framework, an open-source Java full-stack framework that provides the servlet container, JSON-RPC plumbing, connection pooling, and base classes this server extends.
