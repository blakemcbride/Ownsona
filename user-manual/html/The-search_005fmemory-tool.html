5.5 The search_memory tool ¶

Alias of recall with identical inputs, outputs, and errors. Exists because some MCP-client harnesses (notably Claude’s deferred-tool-loading keyword ranker) fail to surface a tool named recall from a generic search query and pick the wrong fallback. Registering the same tool under a more verb-led name gives those harnesses a discoverable handle. Both names dispatch to the same handler.
