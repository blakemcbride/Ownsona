15 Contributing ¶

OwnSona is a single-user system that happens to be open source. Contributions are welcome, with two priorities the project does not trade away:

  • Vendor neutrality on the server side. No generative-LLM calls in the request or write path. The only vendor-coupled seam is EmbeddingProvider. Tool descriptions stay generic.
  • Additive schema migrations. No drops, no renames, no data rewrites inside a migration class. The narrow exception is the embedding-dim resize case documented in Embedding Migration.
