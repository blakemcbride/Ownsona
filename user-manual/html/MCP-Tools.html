5 MCP Tools ¶

OwnSona exposes its functionality as a set of MCP tools. Each one is a JSON-RPC method on the /mcp endpoint, callable by any MCP client that presents a valid OAuth 2.1 access token. This chapter is the reference for every tool’s inputs, outputs, errors, and example calls.
