15.2 Local development ¶

git clone https://github.com/blakemcbride/Ownsona.git
cd Ownsona
cp src/main/backend/application.ini.example src/main/backend/application.ini
$EDITOR src/main/backend/application.ini       # fill in DB + secrets
./bld develop                                   # frontend + backend dev servers

Hot reload covers files under src/main/backend/ (services in Java, Groovy, or Lisp). Files under src/main/core/ (Kiss) and src/main/precompiled/ require a restart of the dev server (or a ./bld -v build + restart).
