7.1 What you need ¶

The same three things you need for any MCP client:

  • Your OwnSona endpoint URL, e.g. https://ownsona.example.com/mcp.
  • Your bearer token (OWNSONA_API_TOKEN from application.ini).
  • Confirmation the server is healthy. A one-line smoke test: 
curl -sS -X POST https://<your-host>/mcp \
    -H "Authorization: Bearer $OWNSONA_API_TOKEN" \
    -H "Content-Type: application/json" \
    -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{}}'

Should return JSON containing "name":"ownsona-mcp". A 401 means the token is wrong; anything else means the server isn’t healthy — fix that first (see Troubleshooting).
